## Описание проекта

Проект посвящён задаче **community detection** (нахождение сообществ) в графах социальных сетей с помощью современных графовых нейронных сетей (Graph Neural Networks).

Используется **Variational Graph Auto-Encoder (VGAE)** из PyTorch Geometric — unsupervised модель, которая:
- Кодирует узлы графа в низкоразмерные эмбеддинги
- Реконструирует рёбра графа
- Полученные эмбеддинги кластеризуются (KMeans или Louvain) для выделения сообществ

Метрики качества:
- **Modularity** — мера качества разбиения на сообщества (~0.38–0.42 на Karate Club)
- **NMI** (Normalized Mutual Information) с ground truth — до 0.95

## Результаты

| Cora (7 классов)      | **0.35–0.42**             | 0.45                          | —                  |
